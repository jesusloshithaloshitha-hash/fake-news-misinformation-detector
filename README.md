# Fake News & Misinformation Detection System

This repository contains the presentation and prototype explanation of an AI-based system designed to detect and block fake news before it is shared on social media.

## Problem
Fake news spreads rapidly due to emotional manipulation and lack of verification, especially in countries with high social media usage.

## Solution
An AI-powered system that analyzes news content before forwarding, detects emotional manipulation and misinformation, and blocks suspicious content with clear user warnings.

## Current Status
This project is presented as a working prototype with a detailed presentation. Code implementation is under development.
